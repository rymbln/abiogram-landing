    Микробиологическое заключение

    Микробиологическое заключение (антибиотикограмма) — это ключевой инструмент для назначения таргетной антимикробной терапии.

    В отличие от многих других видов лабораторно-инструментальных исследований антибиотикограмма прямо указывает лечащему врачу какой препарат (группу препаратов) возможно использовать для терапии, а какой нет. Таким образом, некорректное микробиологическое заключение является ключевой причиной неправильных клинических решений при назначении антимикробной терапии.

    Критически важной задачей является автоматизация рутинных операций микробиологических лабораторий на этапе категоризации результатов определения чувствительности и проверки результатов в соответствии с установленными справочными значениями.

    • Компенсация дефицита кадров со специализированными навыками

    Как работает Справочно-информационная система «Абиограм»?

    Программное обеспечение интегрируется с существующими ЛИС/МИС и микробиологическими анализаторами. Возможно прямое взаимодействие медицинского персонала с интерфейсом ПО. Итог работы ПО – автоматизация процесса категоризации результатов определения чувствительности по исследуемому микроорганизму с оценкой результатов на основе установленных пороговых значений из соответствующих руководств.

    1. Использует информацию о виде возбудителя, результатах определения чувствительности к антибиотикам для данного возбудителя и другую доступную информацию.
    2. Проводит категоризацию результатов на основе установленных критериев.
    3. Осуществляет проверку результата с применением установленных правил.
    4. Формирует ответ, который отправляется в ЛИС/МИС или показывается пользователю.
    Для работы доступна категоризация по руководствам:

    • EUCAST Version 13.0, 2023
    • CLSI M100, 2023
    • EUCAST ECOFF При выходе новых версий руководств производится обновление.
    Содержит более 1100 правил, описанных в соответствующих руководствах (EUCAST, CLSI, ECOFF), которые позволяют:

    • Выявлять необычные и сомнительные результаты
    • Выводить справочные сообщения об индикаторных препаратах
    • Информировать о возможной неэффективности
    • Сообщать о выявлении природной резистентности
